Description of the data set Sudden Infant Death Syndrome (SIDS)

We consider the Sudden Infant Death Syndrome (SIDS) sample data for North Carolina counties, two time periods (1974-78 and 1979-84). Figure a shows the total number of deaths from Sudden Infant Death Syndrome (SIDS) in 1974 for each of the 100 counties in North Carolina. It is for the 100 counties of North Carolina, and includes counts of numbers of live births (also non-white live births) and numbers of sudden infant deaths, for the July 1, 1974 to June 30, 1978 and July 1, 1979 to June 30, 1984 periods.

To load the dataset you can run the following script.

rm(list=ls())
# load data
library(sf)
## Linking to GEOS 3.12.2, GDAL 3.8.5, PROJ 9.3.1; sf_use_s2() is TRUE

                   quiet=TRUE)

The dataset contains the following variables (among others):

  • CNTY_ID: the county ID

  • NAME: The name of the county

  • SID74, SID89: counts of the numbers of sudden infant deaths during 1974-1978 and during 1979-1984 respectively

  • BIR74, BIR79: counts of numbers of live births during 1974-1978 and during 1979-1984 respectively

  • …and geometry settings

                    style = "W" 
                    )
# perform Global Moran’s I hypothesis test 
nc.data.gmoran.test <- moran.test(nc.data.sf$SID74,
                                   nc.listw,

                   quiet=TRUE)

nc.data.sf$logBIR79 <- log(nc.data.sf$BIR79)
# build a neighbours list
library(spdep)
nc.nb <- poly2nb(pl = nc.data.sf )
# supplements a neighbors list with spatial weights for the chosen coding scheme (use style="B" for binary)

                    style = "W"

# fit the SAR model
library(spatialreg)
