Part I Basic stochastic models & related concepts for model building

Note 1. Recall from Section subsec:Aerial-unit-data of “Lecture notes part enu:Handout-1:-Types: Types of spatial data” that modeling aerial unit / lattice data types involves the use of random field models with a discrete index set. Such data are collected over areal units such as pixels, census districts or tomographic bins. Often, there is a natural neighborhood relation or neighborhood structure.

Note 2. This means we need to introduce suitable basic building models able to represent the characteristics of the underline data generating mechanisms. These as the “Discrete Random Fields”.

1 Discrete Random Fields

Note 3. We re-introduce the definition of the random field with regards to the aerial unit data framework.

Definition 4. A random field Z = ( Z s ; s ∈ S ) on a set of indexes S taking values in Z S is a family of random variables { Z s : = Z s ( ω ) ; s ∈ S , ω ∈ Ω } where each Z s ( ω ) is defined on the same probability space ( Ω , F , Pr ) and taking values in Z .

Note 5. In aerial unite data modeling, the (spatial) set of sites S , at which the process is defined, is discrete, it can be finite or infinite (e.g. S ⊆ Z d ), regular (e.g. pixels of an image) or irregular (states of a country).

Note 6. The general state space Z of the random field can be quantitative, qualitative or mixed. E.g., Z = R + in a Gamma random field, Z = N in a Poisson random field, Z = { 0 , 1 } in a binary random field.

Note 7. If Z is finite or countably infinite, the (joint)distribution of Z has a PMF Pr Z ( z ) = Pr ( Z = z ) = Pr ( { Z s = z s ; s ∈ S } ) , ∀ z ∈ Z S otherwise if Z ⊆ R d and Z continuous we will use the joint PDF.

Definition 8. The discrete set of sites S = { s i ; i = 1 , ... , n } is often called lattice of sites.

Notation 9. Often we will use the notation Z s instead of Z ( s ) or Z i instead of Z ( s i ) . Hence, since S = { s i ; i = 1 , ... , n } , we can consider a more convenient notation Z = ( Z s ; s ∈ S ) ⊤ = ( Z i = Z ( s i ) ; i = 1 , ... n ) ⊤ .

Note 10. Modeling aerial unit data often requires the specification of a neighborhood relation or neighborhood structure.

Notation 11. The notation i ∼ j between two sites i , j ∈ S means that “sites i and j are neighboring” according to a “neighborhood relation” ∼ .

Definition 12. Given a lattice of sites S and “neighborhood relation” ∼ , we can define the neighborhood N s of s ∈ S as N s = { s ' ∈ S : s ∼ s ' }
  

Definition 13. Proximity matrix W is called a matrix W which aims at spatially connecting unites i and j in some fashion given some symmetric neighborhood relation ∼ on S . Usually [ W ] i , i = 0 .

Note 14. Proximity matrix W may be such that it represents the neighborhood relation ∼ in a binary fashion e.g. [ W ] i , j = { 1 if  i ∼ j and i ≠ j 0 if  i / ∼ j or i = j or how close site i is to site j based on some distance d ( i , j ) , e.g. [ W ] i , j = { d ( i , j ) if  i ∼ j and i ≠ j 0 if  i / ∼ j or i = j

Note 15. Proximity matrix W does not necessarily need to be symmetric, some times it is standardized as [ W ] i , j ← [ W ] i , j / ∑ j [ W ] i , j .

Example 17. (Logistic/Ising model) Let variable Z i denote the presence of a characteristic as Z i = 1 or absence of it as Z i = 0 on a site labeled by i ∈ S . Then Z = { 0 , 1 } . The Ising model is defined by the (joint) PMF Pr Z ( z ) ∝ exp ( α ∑ i ∈ S z i + β ∑ { i , j } : i ∼ j z i z j ) , ∀ z ∈ Z S (1.1) E.g., it can model a black & white noisy image, where S denotes the labels of the image pixels, and Z i denotes the presence of a black pixel ( Z i = 1 ) or its absence ( Z i = 0 ). Under Ising model (1.1), the characteristic is observed with probability Pr Z i ( z i = 1 ) = exp ( α ) 1 + exp ( α ) when β = 0 . The characteristic’s presence is encouraged in neighboring sites when β > 0 , and discouraged when β < 0 .

Notation 18. We use notation, for A ⊂ S Pr A ( z A | z S \ A ) = Pr ( Z A = z A | Z S \ A = z S \ A )

Definition 19. Local characteristics of a random field Z on S with values in Z are the conditionals Pr i ( z i | z S - i ) = Pr { i } ( z { i } | z S \ { i } ) , i ∈ S , z ∈ Z S
Example 20. (Cont. Example 17) The local characteristics of the Ising model in (1.1) are Pr i ( z i = 1 | z S - i ) = exp ( α + β ∑ { i , j } : i ∼ j z j ) 1 + exp ( α + β ∑ { i , j } : i ∼ j z j )

2 Lattice Random Fields (background)

Note 21. (Recall basic properties Fourier transform) Let { β s : s ∈ S } , S ⊆ Z d be a set of real coefficients
  • The Fourier transform β ˜ ( ω ) = ∑ s ∈ S β s e i s ⊤ ω , ω ∈ ( - π , π ] d .
  • The inverse Fourier transform is β s = ∫ ( - π , π ] d e - i s ⊤ ω β ˜ ( ω ) d ω
  • Regularity conditions for the Fourier transform to be a well-defined function are
    1. If { β s } are summable, ∑ s ∈ S | β s | < ∞ , then β ˜ ( ω ) is bounded and continuous function of ω .
    2. If { β s } are square summable, ∑ s ∈ S | β s | 2 < ∞ then β ˜ ( ω ) is square-integrable over ( - π , π ] d and (visa versa).

Note 22. Let ( Z s : s ∈ S ) be a (weakly) stationary random field where S ⊆ Z d . The following is a tool (similar to Bochner’s theorem) for specifying covariance function of stationary lattice random field. It results from Herglotz’s theorem.

Proposition 23. (Herglotz’s theorem) Let c : Z d → R be a real-valued function on integers for d ≥ 1 . Then c ( ⋅ ) is positive semidefinite (stationary covariance function) if and only if it can be represented as c ( h ) = ∫ ( - π , π ] d exp ( i ω ⊤ h ) d F ( ω ) where F is a symmetric positive bounded finite measure on ( - π , π ] d and F ( - π ) = 0 . F is called spectral measure of c ( h ) . f is called spectral density of c ( h ) if d F ( ω ) = f ( ω ) d ω

Proposition 24. If c ( ⋅ ) is integrable, the spectral density f ( ⋅ ) can be computed by inverse Fourier transformation f ( ω ) = ( 1 2 π ) d ∑ h exp ( - i ω ⊤ h ) c ( h ) d h

Note 25. Let ( Y ( s ) : s ∈ R d ) be a stationary random field with spectral measure F Y ( ω ) , ω ∈ R d and let ( Z s : s ∈ Z d ) with Z s = Y ( s ) for s ∈ Z d , then Z s has spectral measure F Z ( ω ) = ∑ k ∈ Z d F Y ( 2 π k + d ω ) , ω ∈ ( - π , π ] d where frequencies separated by a lag 2 π k , k ∈ Z d , are aliased together in the construction of F Z ( ω ) . Hence, there are infinitely many ways to interpolate a stationary process on Z d to give a stationary random field R d .

3 Compatibility of conditional distributions

Note 27. Here, we discuss how to represent a joint probability distribution via its full conditionals. We need this for model building purposes.

Definition 28. Let random vector Z = ( Z 1 , ... , Z n ) with joint distribution π ( Z 1 , ... , Z n ) . The set of distributions { π i ( ⋅ | Z - i ) ; i = 1 , ... n } is called compatible to the joint distribution π ( Z 1 , ... , Z n ) if the joint distribution π ( Z 1 , ... , Z n ) has conditionals { π i ( Z i | Z - i ) ; i = 1 , .. , n } .

Note 29. To specify suitable building models representing spatial dependency of a random field ( Z i ) i ∈ S , it is often easier to visualize the joint distribution Pr z in terms of conditional distributions { π i ( Z i | Z S - i ) ; i ∈ S } rather than directly.

Note 30. Thus, instead of specifying a joint model for ( Z i ) i ∈ S , a researcher may propose putative families of conditional distributions { π i ( Z i | Z S - i ) ; i ∈ S } . However, an arbitrary chosen set of conditional distributions { π i ( ⋅ | ⋅ ) ; i ∈ S } is not generally compatible, in the sense that there exists a joint distribution for ( Z i ) i ∈ S , and hence we need to impose conditions.

Note 31. In what follows, we discuss necessary and sufficient conditions regarding compatibility.

Proposition 32. (Compatibility condition) Let F be a joint distribution with d F ( x , y ) = f ( x , y ) d ( x , y ) on S x × S y .   2. there exist functions u and v where g ( x | y ) / q ( y | x ) = u ( x ) v ( y ) for all ( x , y ) ∈ N and ∫ u ( x ) d x < ∞

4 Spatial autoregressive models

We present two basic spatial Autoregressive models, the SAR and CAR, able to represent spatial dependency.

4.1 Simultaneous AutoRegressive (SAR) models


Definition 39. “Autoregressive” representation for lattice random field ( Z s ; s ∈ S ) , S ⊆ Z d is called ∑ h ∈ H a h ( Z s + h - μ s + h ) = ε s , s ∈ S (4.1) where { a h ; h ∈ H } , H = { h ∈ Z d : s + h ∈ S } , are coefficients and { ε s ; s ∈ S } is a discrete white noise random field with variance λ s = Var ( ε s ) .

4.1.1 Assuming stationarity

Note 40. We will focus our study on stationary CAR random field ( Z s ; s ∈ S ) , i.e. μ s = μ and λ s = σ ε 2 for s ∈ S ⊆ Z d .

Note 41. { ε s ; s ∈ S } has c.f. c ε ( h ) = σ ε 2 δ { 0 } ( h ) and hence spectral density f ε ( ω ) = σ ε 2 / ( 2 π ) d . The spectral density f for ( Z s ; s ∈ S ) is f ( ω ) = 1 | a ˜ ( ω ) | 2 f ε ( ω ) = 1 | a ˜ ( ω ) | 2 ( 1 2 π ) d ∫ e - i ω ⊤ h c ε ( h ) d h (4.2) = 1 | a ˜ ( ω ) | 2 ( 1 2 π ) d ∑ h ∈ Z d e - i ω ⊤ h σ ε 2 δ { 0 } ( h ) d h = 1 | a ˜ ( ω ) | 2 σ ε 2 ( 2 π ) d where a ˜ ( ω ) = ∑ h a h e i h ⊤ ω since (26).

Note 42. For random field ( Z s ; s ∈ S ) to be stationary, f ( ω ) in (4.2) must be integrable function and bounded function on ω ∈ ( - π , π ] d . Hence, we can set restrictions on coefficients a 0 > 0 , and ∑ h | a h | < a 0 (4.3) satisfying regularity conditions in (21).

Note 43. To make model (4.1) identifiable and use it for inference, we can introduce further restrictions a h = a - h .

Definition 44. Lattice random field ( Z s ; s ∈ S ) is called Simultaneous AutoRegressive (SAR) if it is given in an “Autoregression” representation (4.1) ∑ h ∈ N a h Z s + h = ε s , s ∈ S whose coefficients { a h ; h ∈ H } satisfy the symmetry condition a h = a - h , ∀ h and f Z ( ω ) in (4.2) is an integrable function over ω ∈ ( - π , π ] d .

4.3 Conditional autoregressive models (CAR)

Definition 48. A lattice random field ( Z s ; s ∈ S ) , S ⊆ Z d is called Conditional AutoRegressive (CAR) model if E ( Z s | Z S - s ) = μ s + ∑ h ∈ H b h ( Z s + h - μ s + h ) Var ( Z s | Z S - s ) = κ s where b h = - b h and H = { h ∈ Z d : s + h ∈ S } .

Note 49. Alternatively CAR lattice random field ( Z s ; s ∈ S ) , S ⊆ Z d can be written as Z s = μ s + ∑ h ∈ H b h ( Z s + h - μ s + h ) + ϵ s (4.5) where ( ϵ s : s ∈ S ) is the residual random field with mean E ( ϵ s ) = 0 and variance Var ( ϵ s ) = κ s . Also b h = - b h and H = { h ∈ Z d : s + h ∈ S } .
