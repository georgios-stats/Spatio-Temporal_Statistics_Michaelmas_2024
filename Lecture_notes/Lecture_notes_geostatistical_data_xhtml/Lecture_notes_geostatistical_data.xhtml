Part I Basic stochastic models & related concepts for model building

Note 1. We discuss basic stochastic models and concepts for modeling point referenced data in the Geostatistics framework.

1 Stochastic processes (or Random fields)

Definition 2. A stochastic process (or random field) Z = ( Z s ; s ∈ S ) taking values in Z ⊆ R q , q ≥ 1 is a family of random variables { Z s : = Z s ( ω ) ; s ∈ S , ω ∈ Ω } defined on the same probability space ( Ω , F , Pr ) and taking values in Z . The label s ∈ S is called site, the set S ⊆ R d is called the (spatial) set of sites at which the process is defined, and Z is called the state space of the process.

Note 3. Given a set { s 1 , ... , s n } of sites, with s i ∈ S , the random vector ( Z ( s 1 ) , ... , Z ( s n ) ) ⊤ has a well-defined probability distribution that is completely determined by its joint CDF F s 1 , ... , s n ( z 1 , ... , z n ) = Pr ( Z ( s 1 ) ≤ z 1 , ... , Z ( s n ) ≤ z n ) Finite dimensional distributions (or fidi’s) of Z is called the ensemble of all such joint CDF’s with n ∈ N and { s i ∈ S } .

Note 4. According to Kolmogorov Theorem 5, to define a random field model, one must specify the joint distribution of ( Z ( s 1 ) , ... , Z ( s n ) ) ⊤ for all of n and all { s i ∈ S } i = 1 n in a consistent way.
Proposition 5. (Kolmogorov consistency theorem) Let Pr s 1 , ... , s n be a probability on R n with join CDF F s 1 , ... , s n for every finite collection of points s 1 , ... , s n . If F s 1 , ... , s n is symmetric w.r.t. any permutation p F p ( s 1 ) , ... , p ( s n ) ( z p ( 1 ) , ... , z p ( n ) ) = F s 1 , ... , s n ( z 1 , ... , z n ) for all n ∈ N , { s i ∈ S } , and { z ∈ R } , and all if all permulations p are consistent in the sense lim z n → ∞ F s 1 , ... , s n ( z 1 , ... , z n ) = F s 1 , ... , s n - 1 ( z 1 , ... , z n - 1 ) or all n ∈ N , { s i ∈ S } , and { z ∈ R } , then there exists a random field Z whose fidi’s coincide with those in F .

Example 6. Let n ∈ N , let { f i : T → R ;   i = 1 , ... , n } be a set of constant functions, and let { Z i ∼ N ( 0 , 1 ) } i = 1 n be a set of independent random variables. Then Z ˜ s = ∑ i = 1 n Z i f i ( s ) , s ∈ S (1.1) is a well defined stochastic process as it satisfies Theorem 5.

2 Second order processes (or Second order random fields)

Note 14. We introduce a particular class of stochastic processes whose mean and covariance functions exist and which can be used of spatial data modeling.

Definition 15. Second order process (or second order random field) Z = ( Z s ; s ∈ S ) is called the stochastic process where E ( Z s 2 ) < ∞ for all s ∈ S .

Example 16. In second order processes ( Z s ) s ∈ S , then the associated mean function μ ( ⋅ ) and covariance function c ( ⋅ , ⋅ ) exist, because c ( s , t ) = E ( Z s Z t ) - E ( Z s ) E ( Z t ) for s , t ∈ S .

Z = ( Z s ; s ∈ S ) indexed by S ⊆ R d is a Gaussian process (GP) or random field (GRF) if for any n ∈ N and for any finite set { s 1 , ... , s n ; s i ∈ S } , the random vector ( Z s 1 , ... , Z s n ) ⊤ follows a multivariate normal distribution.
Proposition 19. A GP Z = ( Z s ; s ∈ S ) is fully characterized by its mean function μ : S → R with μ ( s ) = E ( Z s ) , and its covariance function with c ( s , s ' ) = Cov ( Z s , Z s ' ) .

Notation 20. Hence, we denote the GP as Z ( ⋅ ) ∼ G P ( μ ( ⋅ ) , c ( ⋅ , ⋅ ) ) .

Note 21. When using GP for spatial modeling we may need to specify its functional parameters i.e. the mean and covariance functions.

Note 22. An popular form of mean functions are polynomial expansions, such as μ ( s ) = ∑ j = 0 p - 1 β j s j for some tunable unknown parameter β . An popular form of covariance functions (c.f.), for some tunable unknown parameter β , σ 2 , are
  1. Exponential c.f. c ( s , s ' ) = σ 2 exp ( - β ∥ s - s ' ∥ 1 )
  2. Gaussian c.f. c ( s , s ' ) = σ 2 exp ( - β ∥ s - s ' ∥ 2 2 )
  3. Nugget c.f. c ( s , s ' ) = σ 2 1 ( s = s ' )

Definition 30. A random field Z = ( Z s ) s ∈ R d is weakly stationary (or second order stationary) if, for all s , s ' ∈ R d ,
  1. E ( Z s 2 ) < ∞ (finite)
  2. E ( Z s ) = m (constant)
  3. Cov ( Z s , Z s ' ) = c ( s ' - s ) for some even function c : R d → R (lag dependency)

Definition 31. Weakly (or second order) stationary covariance function is called the c.f. of a weakly stationary stochastic process.
