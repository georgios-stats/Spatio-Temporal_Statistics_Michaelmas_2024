Part I Basic stochastic models & related concepts for model building

Note 1. We discuss basic stochastic models and concepts for modeling point referenced data in the Geostatistics framework.

1 Random fields (or Stochastic processes)

Definition 2. A random field (or stochastic process, or random function) Z = ( Z ( s ) ; s ∈ S ) taking values in Z ⊆ R q , q ≥ 1 is a family of random variables { Z ( s ) : = Z ( s ; ω ) ; s ∈ S , ω ∈ Ω } defined on the same probability space ( Ω , F , Pr ) and taking values in Z . The label s ∈ S is called site, the set S ⊆ R d is called the (spatial) set of sites at which the random field is defined, and Z is called the state space of the field.

Note 3. Given a set of sites { s 1 , ... , s n } , with s i ∈ S and n ∈ N , the random vector ( Z ( s 1 ) , ... , Z ( s n ) ) ⊤ has a well-defined probability distribution that is completely determined by its joint CDF F s 1 , ... , s n ( z 1 , ... , z n ) : = Pr ( Z ( s 1 ) ≤ z 1 , ... , Z ( s n ) ≤ z n ) The family of all finite-dimensional distributions (or fidi’s) of Z is called the spatial distribution of the process .

Note 4. According to Kolmogorov Theorem 5, to define a random field model, one must specify the joint distribution of ( Z ( s 1 ) , ... , Z ( s n ) ) ⊤ for all of n and all { s i ∈ S ; i = 1 , ... , n } in a consistent way.
Proposition 5. (Kolmogorov consistency theorem) Let Pr s 1 , ... , s n be a probability on R n with join CDF F s 1 , ... , s n for every finite collection of points s 1 , ... , s n . If F s 1 , ... , s n is symmetric w.r.t. any permutation p F p ( s 1 ) , ... , p ( s n ) ( z p ( 1 ) , ... , z p ( n ) ) = F s 1 , ... , s n ( z 1 , ... , z n ) for all n ∈ N , { s i ∈ S } , and { z ∈ R } , and all if all permulations p are consistent in the sense lim z n → ∞ F s 1 , ... , s n ( z 1 , ... , z n ) = F s 1 , ... , s n - 1 ( z 1 , ... , z n - 1 ) or all n ∈ N , { s i ∈ S } , and { z i ∈ R } , then there exists a random field Z whose fidi’s coincide with those in F .

Example 6. Let n ∈ N , let { X i : S → R ;   i = 1 , ... , n } be a set of constant functions, and let { Z i ∼ N ( 0 , 1 ) } i = 1 n be a set of independent random variables. Then Z ˜ ( s ) = ∑ i = 1 n Z i X i ( s ) , s ∈ S (1.1) is a well defined random field as it satisfies Theorem 5.

2 Second order random fields (or Second order processes)

Note 13. We introduce a particular class of random fields whose mean and covariance functions exist and which can be used for spatial data modeling.

Definition 14. Second order random field (or second order process) ( Z ( s ) ; s ∈ S ) is called the random field where E ( ( Z ( s ) ) 2 ) < ∞ for all s ∈ S .

Example 15. In second order random field ( Z ( s ) ; s ∈ S ) the associated mean function μ ( ⋅ ) and covariance function c ( ⋅ , ⋅ ) exist, because c ( s , t ) = E ( Z ( s ) Z ( t ) ) - E ( Z ( s ) ) E ( Z ( t ) ) for s , t ∈ S .

4 Strong stationarity

Note 24. We introduce a specific behavior of random field to build our models.

Notation 25. Formally, we define the separation (or lag) set as H = { h ∈ R d : s ∈ S , s + h ∈ S } where S ⊆ R d is the spatial domain for d = 1 , 2 , 3 , ... . However, we will consider cases where S = R d and H = R d for d = 1 , 2 , 3 , ... in Euclidean spaces.
Definition 26. A random field ( Z ( s ) ; s ∈ S ) is strongly stationary on S if for all finite sets consisting of s 1 , ... , s n ∈ S , n ∈ N , for all k 1 , ... , k n ∈ R , and for all h ∈ H Pr ( Z ( s 1 + h ) ≤ k 1 , ... , Z ( s n + h ) ≤ k n ) = Pr ( Z ( s 1 ) ≤ k 1 , ... , Z ( s n ) ≤ k n )

Note 27. Yuh... strong stationary may represent a behavior being too “restrictive” to be used for spatial data modeling as it is able to represent only limiting number of spatial dependencies.

5 Weak stationarity (or second order stationarity)

Note 28. We introduce another weaker behavior of random field able to represent a larger class of spatial dependencies.

Note 29. Instead of working with the “restrictive” strong stationarity, we could just properly specify the behavior of the first two moments only; notice that Definition 26 implies that, given E ( ( Z ( s ) ) 2 ) < ∞ , it is E ( Z ( s ) ) = E ( Z ( s + h ) ) = contst ... and Cov ( Z ( s ) , Z ( s ' ) ) = Cov ( Z ( s + h ) , Z ( s ' + h ) ) = h = - s ' Cov ( Z ( s - s ' ) , Z ( 0 ) ) = funct of lag ...

Definition 30. A random field ( Z ( s ) ; s ∈ S ) is called stationary random field (s.r.f.) (or weakly stationary or second order stationary) if it has constant mean and translation invariant covariance; i.e. for all s , s ' ∈ R d ,
  1. E ( ( Z ( s ) ) 2 ) < ∞ (finite)
  2. E ( Z ( s ) ) = μ (constant)
  3. Cov ( Z ( s ) , Z ( s ' ) ) = c ( s ' - s ) for some even function c : S → R (lag dependency)

Definition 31. Stationary (or weakly or second order stationary) covariance function is called the c.f. of a stationary random field.

6 Covariogram

Note 32. We introduce the covariogram function able to express many aspects of the behavior of a (weakly) stationary random field and hence be used as statistical descriptive tool.

Definition 33. The covariogram function of a weakly stationary random field ( Z ( s ) ; s ∈ S ) is defined by c : H → R with c ( h ) = Cov ( Z ( s ) , Z ( s + h ) ) , ∀ s ∈ S , ∀ h ∈ H .

Example 34. For the Gaussian c.f. c ( s , t ) = σ 2 exp ( - ϕ ∥ s - t ∥ 2 2 ) in (Ex. 20(2.)), we may denote just c ( h ) = c ( s , s + h ) = σ 2 exp ( - ϕ ∥ h ∥ 2 2 ) (6.1) Observe that, in Figures 3.1 &3.2, the smaller the ϕ , the smoother the realization (aka slower changes). One way to justify this observation is to think that smaller values of ϕ essentially bring the points closer by re-scaling spatial lags h in the c.f.

Proposition 35. If c : H → R is the covariogram of a weakly stationary random field ( Z ( s ) ; s ∈ S ) then:
  1. c ( h ) = c ( - h ) for all h ∈ H
  2. | c ( h ) | ≤ c ( 0 ) = Var ( Z ( s ) ) for all h ∈ H
  3. c ( ⋅ ) is semi-positive definite; i.e. for all n ∈ N - { 0 } , ( a 1 , ... , a n ) ∈ R n , and ( s 1 , ... , s n ) ⊆ S n ∑ i = 1 n ∑ j = 1 n a i a j c ( s i - s j ) ≥ 0

Note 36. Given there is some knowledge of the characteristic functions of a suitable distribution, the following spectral representation theorem helps in the specification of a suitable covariogram.

Theorem 37. (Bochner’s theorem) Let c : R d → R be a continuous even real-valued function for d ≥ 1 . Then c ( ⋅ ) is positive semidefinete (hence a covariogram of a stationary random field) if and only if it can be represented as c ( h ) = ∫ R d exp ( i ω ⊤ h ) d F ( ω ) where F is a symmetric positive finite measure on R d called spectral measure.

Note 38. In our course, we focus on cases where F has a density f ( ⋅ ) i.e. d F ( ω ) = f ( ω ) d ω . f ( ⋅ ) is called spectral density of c ( ⋅ ) , it is c ( h ) = ∫ R d exp ( i ω ⊤ h ) f ( ω ) d ω , and it dies as lim | h | → ∞ c ( h ) = 0

Theorem 39. If c ( ⋅ ) is integrable, the spectral density f ( ⋅ ) can be computed by inverse Fast Fourier transformation f ( ω ) = ( 1 2 π ) d ∫ R d exp ( - i ω ⊤ h ) c ( h ) d h
Example 40. Consider the Gaussian c.f. c ( h ) = σ 2 exp ( - ϕ ∥ h ∥ 2 2 ) for σ 2 , β > 0 and h ∈ R d . Then, by using Theorem 37, the spectral density is f ( ω ) = ( 1 2 π ) d ∫ R d exp ( - i ω ⊤ h ) σ 2 exp ( - ϕ ∥ h ∥ 2 2 ) d h = σ 2 ( 1 2 π ) d ∏ j = 1 d ∫ R exp ( - i ω j h j - ϕ h j 2 ) d h = σ 2 ( 1 2 π ) d ∏ j = 1 d ∫ R exp ( - ϕ ( h j - ( - i ω / ( 2 ϕ ) ) ) 2 ) exp ( - ω j 2 / ( 4 ϕ ) ) d h j = σ 2 ( 1 4 π ϕ ) d / 2 exp ( - ∥ ω ∥ 2 2 / ( 4 ϕ ) ) i.e. it has a Gaussian form.

Definition 41. Let ( Z ( s ) : s ∈ S ) be a weakly stationary random field with covariogram function c : H → R and c ( h ) = Cov ( Z ( s ) , Z ( s + h ) ) . The correlogram function ρ : H → [ - 1 , 1 ] is defined as ρ ( h ) = c ( h ) c ( 0 ) .

7 Intrinsic stationarity (of order zero)

Note 42. (Weakly) stationary random fields may not be sufficiently general for all of our applications. This class of random fields can be extended by considering intrinsic stationary instead. For instance in certain applications it has been noticed that the “process” we wish to model has increments whose variance Var ( Z ( s + h ) - Z ( s ) ) = Var ( Z ( s + h ) ) + Var ( Z ( s ) ) - 2 Cov ( Z ( s + h ) , Z ( s ) ) increases indefinitely with | h | ; this “process” cannot be modeled in the class of (weakly) stationary random fields whose increments have bounded variance Var ( Z ( s + h ) - Z ( s ) ) = 2 ( c ( 0 ) - c ( h ) ) < 2 c ( 0 ) .

Definition 43. A random field ( Z ( s ) : s ∈ S ) is called intrinsic random field (i.r.f.) (or intrinsic stationary r.f.) if, for all h ∈ H , ,
  1. E ( Z ( s + h ) - Z ( s ) ) 2 < ∞
  2. E ( Z ( s + h ) - Z ( s ) ) = μ ( h ) for some function μ : H → R (lag dependent)
  3. Var ( Z ( s + h ) - Z ( s ) ) = 2 γ ( h ) for some function γ : H → R (lag dependent)

Definition 44. Intrinsic covariance function (i.c.f.) is called the c.f. of an intrinsically stationary stochastic process.

Example 45. The random field with covariance function c ( s , t ) = 1 2 ( ∥ s ∥ 2 H + ∥ t ∥ 2 H - ∥ t - s ∥ 2 H ) , H ∈ ( 0 , 1 ) is not stationary r.f. because c ( s , s + h ) = 1 2 ( ∥ s ∥ 2 H + ∥ s + h ∥ 2 H - ∥ h ∥ 2 H ) for h ∈ H but it intrinsic r.f. because 1 2 Var ( Z ( s + h ) - Z ( s ) ) = 1 2 ( Var ( Z ( s ) ) + Var ( Z ( s + h ) ) - 2 Cov ( Z ( s ) , Z ( s + h ) ) ) = 1 2 ∥ h ∥ 2 H

8 Incremental mean function

Definition 46. Incremental mean function of the intrinsic random field ( Z ( s ) : s ∈ S ) is defined as μ : H → R with μ ( h ) = E ( Z ( s + h ) - Z ( s ) ) .

9 Semivariogram

Note 47. A very informative tool about the behavior of the intrinsic random field is the semivariogram function defined below.
Definition 48. The semivariogram of an intrinsic random field ( Z ( s ) : s ∈ S ) is defined as γ : H → R , with γ ( h ) = 1 2 Var ( Z ( s + h ) - Z ( s ) )

Definition 49. Variogram of an intrinsic random field ( Z ( s ) : s ∈ S ) is called the quantity 2 γ ( h ) .

Note 50. A stationary random field with covariogram c ( ⋅ ) and mean μ is intrinsic stationary as well with semivariogram γ ( h ) = c ( 0 ) - c ( h ) , (9.1) and constant incremental mean μ ( h ) = μ .
Example 51. For the Gaussian covariance function (Ex. 34) the semivariogram is γ ( h ) = c ( 0 ) - c ( h ) = σ 2 ( 1 - exp ( - β ∥ h ∥ 2 2 ) )

Proposition 52. Properties of semivariogram. Let ( Z ( s ) : s ∈ S ) be an intrinsic random field, then
  1. It is γ ( h ) = γ ( - h ) , γ ( h ) ≥ 0 , and γ ( 0 ) = 0
  2. Semivariogram is conditionally negative definite (c.n.d.): if for all n ∈ N , ( a 1 , ... , a n ) ⊆ R n s.t. ∑ i = 1 n a i = 0 , and for all ( s 1 , ... , s n ) ⊆ S n , it is ∑ i = 1 n ∑ j = 1 n a i a j γ ( s i - s j ) ≤ 0

Note 54. A semivariogram tends to be an increasing function of the lag ∥ h ∥ . Recall that for weakly stationary random fields with c.f. c ( ⋅ ) , it is γ ( h ) = c ( 0 ) - c ( h ) where common logic suggests that c ( h ) is decreases with ∥ h ∥ .

Note 55. If γ ( h ) is a positive constant for all non-zero lags h ≠ 0 , then Z ( s 1 ) and Z ( s 2 ) are uncorrelated regardless of how close s 1 and s 2 are. Then Z ( ⋅ ) is called white noise.

Note 56. Conversely, a non zero slope of the variogram indicates some structure.
Nugget Effect
Note 57. Nugget effect is the semivariagram limiting value σ ε 2 = lim ∥ h ∥ → 0 γ ( h ) when σ ε 2 ≠ 0 .

Note 58. When used for modeling, nugget effect σ ε 2 ≠ 0 may expresses (1) measurement errors (e.g., if we collect repeated measurements at the same location s ) or (2) some microscale variation causing discontinuity in the origin that cannot be detected from the data i.e. the spatial gaps because we collect a finite set of measurements at spatial locations. Ideally, a more detailed decomposition σ ε 2 = σ MS 2 + σ MS 2 can be considered where σ MS 2 refers to the microscale and σ MS 2 refers to the measurement error. However this may lead to non-identifiability, without any obvious tweak to address it.

Sill
Definition 59. Sill is the semivariagram limiting value lim ∥ h ∥ → ∞ γ ( h ) .

Note 60. For intrinsic processes, the sill may be infinite or finite. For weakly random field, the sill is always finite.
Partial sill  
Definition 61. Partial sill is lim ∥ h ∥ → ∞ γ ( h ) - lim ∥ h ∥ → 0 γ ( h ) which takes into account the nugget.
Range  
Note 62. Range is the distance at which the semivariogram reaches the Sill. It can be infinite or finite.
Other

Note 63. An abrupt change in slope indicates the passage to a different structuration of the values in space. This is often modeled via decomposition of processes with different semivariograms. E.g., let independent random fields Y ( ⋅ ) and X ( ⋅ ) with different semivariograms γ Y and γ X , then random field Z ( ⋅ ) with Z ( s ) = Y ( s ) + X ( s ) has semivariogram γ Z ( h ) = γ Y ( h ) + γ X ( h ) which may present such a behavior.

11 Isotropy

Note 64. Isotropy introduses the assumption of “rotation invariance”.

Note 65. Isotropy applies to both intrinsic and (weakly) stationary random fields.

Definition 66. An intrinsic random field ( Z ( s ) : s ∈ S ) is isotropic iff ∀ s , t ∈ S , 1 2 Var ( Z ( s ) - Z ( t ) ) = γ ( ∥ t - s ∥ ) , for some function  γ: R + → R . (11.1)

Definition 67. Isotropic semivariogram γ : H → R is the semivariogram of the isotropic random field (sometimes for simplicity of notation we use γ : R + → R with γ ( ∥ h ∥ ) = 1 2 Var ( Z ( s ) - Z ( s - h ) ) .

Definition 68. Isotropic covariance function C : S × S → R is called the covariance function satisfying (11.1).

Definition 69. Isotropic covariogram c : H → R of a weakly stationary process is the covariogram associated to an isotropic semivariogram. Sometimes for simplicity of notation we use c : R + → R with c ( ∥ h ∥ ) from (11.1).

12 Anisotropy

Note 74. Dependence between Z ( s ) and Z ( s + h ) is a function of both the magnitude and the direction of separation h . This can be caused by the underlying physical process evolving differently in space (e.g., vertical and horizontal axes).
Definition 75. The semivariogram γ : H → R is anisotropic if there are h 1 and h 2 with same length ∥ h 1 ∥ = ∥ h 2 ∥ but different direction h 1 / ∥ h 1 ∥ ≠ h 2 / ∥ h 2 ∥ that produce different semivariograms γ ( h 1 ) ≠ γ ( h 2 ) .

Definition 76. The intrinsically random field ( Z ( s ) : s ∈ S ) is anisotropic if its semivariogram is anisotropic.

Definition 77. The covariogram c : H → R is anisotropic if there are h 1 and h 2 with same length ∥ h 1 ∥ = ∥ h 2 ∥ but different direction h 1 / ∥ h 1 ∥ ≠ h 2 / ∥ h 2 ∥ that produce different covariogram c ( h 1 ) ≠ c ( h 2 ) .

Definition 78. The (weakly) stationary random field ( Z ( s ) : s ∈ S ) is anisotropic if its covariogram is anisotropic.
