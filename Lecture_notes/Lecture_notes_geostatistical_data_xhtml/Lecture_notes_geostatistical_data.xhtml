Part I Basic stochastic models & related concepts for model building

Note 1. We discuss basic stochastic models and concepts for modeling point referenced data in the Geostatistics framework.

1 Random fields (or Stochastic processes)

Definition 2. A random field (or stochastic process, or random function) Z = ( Z ( s ) ; s ∈ S ) taking values in Z ⊆ R q , q ≥ 1 is a family of random variables { Z ( s ) : = Z ( s ; ω ) ; s ∈ S , ω ∈ Ω } defined on the same probability space ( Ω , F , Pr ) and taking values in Z . The label s ∈ S is called site, the set S ⊆ R d is called the (spatial) set of sites at which the random field is defined, and Z is called the state space of the field.

Note 3. Given a set of sites { s 1 , ... , s n } , with s i ∈ S and n ∈ N , the random vector ( Z ( s 1 ) , ... , Z ( s n ) ) ⊤ has a well-defined probability distribution that is completely determined by its joint CDF F s 1 , ... , s n ( z 1 , ... , z n ) : = Pr ( Z ( s 1 ) ≤ z 1 , ... , Z ( s n ) ≤ z n ) The family of all finite-dimensional distributions (or fidi’s) of Z is called the spatial distribution of the process .

Note 4. According to Kolmogorov Theorem 5, to define a random field model, one must specify the joint distribution of ( Z ( s 1 ) , ... , Z ( s n ) ) ⊤ for all of n and all { s i ∈ S ; i = 1 , ... , n } in a consistent way.
Proposition 5. (Kolmogorov consistency theorem) Let Pr s 1 , ... , s n be a probability on R n with join CDF F s 1 , ... , s n for every finite collection of points s 1 , ... , s n . If F s 1 , ... , s n is symmetric w.r.t. any permutation p F p ( s 1 ) , ... , p ( s n ) ( z p ( 1 ) , ... , z p ( n ) ) = F s 1 , ... , s n ( z 1 , ... , z n ) for all n ∈ N , { s i ∈ S } , and { z ∈ R } , and all if all permulations p are consistent in the sense lim z n → ∞ F s 1 , ... , s n ( z 1 , ... , z n ) = F s 1 , ... , s n - 1 ( z 1 , ... , z n - 1 ) or all n ∈ N , { s i ∈ S } , and { z i ∈ R } , then there exists a random field Z whose fidi’s coincide with those in F .

Example 6. Let n ∈ N , let { X i : S → R ;   i = 1 , ... , n } be a set of constant functions, and let { Z i ∼ N ( 0 , 1 ) } i = 1 n be a set of independent random variables. Then Z ˜ ( s ) = ∑ i = 1 n Z i X i ( s ) , s ∈ S (1.1) is a well defined random field as it satisfies Theorem 5.

2 Second order random fields (or Second order processes)

Note 13. We introduce a particular class of random fields whose mean and covariance functions exist and which can be used for spatial data modeling.

Definition 14. Second order random field (or second order process) ( Z ( s ) ; s ∈ S ) is called the random field where E ( ( Z ( s ) ) 2 ) < ∞ for all s ∈ S .

Example 15. In second order random field ( Z ( s ) ; s ∈ S ) the associated mean function μ ( ⋅ ) and covariance function c ( ⋅ , ⋅ ) exist, because c ( s , t ) = E ( Z ( s ) Z ( t ) ) - E ( Z ( s ) ) E ( Z ( t ) ) for s , t ∈ S .

4 Strong stationarity

Note 24. We introduce a specific behavior of random field to build our models.

Notation 25. Formally, we define the separation (or lag) set as H = { h ∈ R d : s ∈ S , s + h ∈ S } where S ⊆ R d is the spatial domain for d = 1 , 2 , 3 , ... . However, we will consider cases where S = R d and H = R d for d = 1 , 2 , 3 , ... in Euclidean spaces.
Definition 26. A random field ( Z ( s ) ; s ∈ S ) is strongly stationary on S if for all finite sets consisting of s 1 , ... , s n ∈ S , n ∈ N , for all k 1 , ... , k n ∈ R , and for all h ∈ H Pr ( Z ( s 1 + h ) ≤ k 1 , ... , Z ( s n + h ) ≤ k n ) = Pr ( Z ( s 1 ) ≤ k 1 , ... , Z ( s n ) ≤ k n )

Note 27. Yuh... strong stationary may represent a behavior being too “restrictive” to be used for spatial data modeling as it is able to represent only limiting number of spatial dependencies.

Note 28. We introduce another weaker behavior of random field able to represent a larger class of spatial dependencies.

Definition 30. A random field ( Z ( s ) ; s ∈ S ) is called stationary random field (s.r.f.) (or weakly stationary or second order stationary) if it has constant mean and translation invariant covariance; i.e. for all s , s ' ∈ R d ,
  1. E ( ( Z ( s ) ) 2 ) < ∞ (finite)
  2. E ( Z ( s ) ) = μ (constant)
  3. Cov ( Z ( s ) , Z ( s ' ) ) = c ( s ' - s ) for some even function c : S → R (lag dependency)

Definition 31. Stationary (or weakly or second order stationary) covariance function is called the c.f. of a stationary random field.

7 Intrinsic stationarity (of order zero)

Note 42. The class of (weakly) stationary random fields may not be sufficiently general enough to model a large number of important applications. E.g., in certain applications, it has been noticed that the “underline process” we wish to model presents increments whose variance Var ( Z ( s + h ) - Z ( s ) ) = Var ( Z ( s + h ) ) + Var ( Z ( s ) ) - 2 Cov ( Z ( s + h ) , Z ( s ) ) increases indefinitely with | h | ; this “process” cannot be modeled within the class of (weakly) stationary random fields whose increments have bounded variance Var ( Z ( s + h ) - Z ( s ) ) = 2 ( c ( 0 ) - c ( h ) ) < 2 c ( 0 ) . Intrinsic stationary is a weaker assumption extending the class of models we can use.

Definition 43. A random field ( Z ( s ) : s ∈ S ) is called intrinsic random field (i.r.f.) (or intrinsic stationary r.f.) if, for all h ∈ H , ,

Note 60. Conversely, a non zero slope of the variogram indicates some structure.
Nugget Effect
Note 61. Nugget effect is the semivariagram limiting value σ ε 2 = lim ∥ h ∥ → 0 γ ( h ) when σ ε 2 ≠ 0 .

Note 62. When used for modeling, nugget effect σ ε 2 ≠ 0 may expresses (1) measurement errors (e.g., if we collect repeated measurements at the same location s ) or (2) some microscale variation causing discontinuity in the origin that cannot be detected from the data i.e. the spatial gaps because we collect a finite set of measurements at spatial locations. Ideally, a more detailed decomposition σ ε 2 = σ MS 2 + σ MS 2 can be considered where σ MS 2 refers to the microscale and σ MS 2 refers to the measurement error. However this may lead to non-identifiability, without any obvious tweak to address it.

Sill
Definition 63. Sill is the semivariagram limiting value lim ∥ h ∥ → ∞ γ ( h ) .

Note 64. For intrinsic processes, the sill may be infinite or finite. For weakly random field, the sill is always finite.
Partial sill  
Definition 65. Partial sill is lim ∥ h ∥ → ∞ γ ( h ) - lim ∥ h ∥ → 0 γ ( h ) which takes into account the nugget.
Range  
Note 66. Range is the distance at which the semivariogram reaches the Sill. It can be infinite or finite.
Other

Note 67. An abrupt change in slope indicates the passage to a different structuration of the values in space. This is often modeled via decomposition of processes with different semivariograms. E.g., let independent random fields Y ( ⋅ ) and X ( ⋅ ) with different semivariograms γ Y and γ X , then random field Z ( ⋅ ) with Z ( s ) = Y ( s ) + X ( s ) has semivariogram γ Z ( h ) = γ Y ( h ) + γ X ( h ) which may present such a behavior.

14.1 Linear Model of Regionalization

Note 101. A spatial phenomenon can be thought as being the sum of several independent subphenomena acting at different characteristic scales. A linear model can be set up to split the stochastic process representing the phenomenon into several uncorrelated stochastic processes, each with a different variogram or covariance function and characterizing different aspect of the overall phenomenon under investigation.

Note 102. μ ( s ) = E ( Z ( s ) )
is the deterministic mean (or drift) structure. It aims to represent the “large scale variation”.
W ( s )
is a zero mean second order continuous intrinsic random field whose range is larger than gaps between the sites (sampling grid). It aims to represent “smooth small scale variation”.
η ( s )
is a zero mean intrinsic random field whose variogram range exists and is smaller than the gaps between the sites. It aims to represent “microscale variation”
ε ( s )
is a zero-mean white-noise process (modeled as nugget effect). It aims to represent “measurement error or noise”
W ( s ) , η ( s ) , ε ( s ) are mutually independent.
